# OIBSIP
A Java-based console application for train ticket booking with PNR, passenger details, and cancellation System.
⚙️ Features
# Login System
Login ID must be 5–12 characters (letters, digits, underscores allowed).
Password must be 8–15 characters with uppercase, lowercase, digit, and a special character.
# Reservation System
Enter passenger details (Name, Age, Train Number, Class, Date, Source, Destination).
Validates train number from available train list.
Generates a unique PNR number for every booking.
Stores reservation details in memory using a HashMap.
# Cancellation System
Cancel a booking using PNR number.
Confirms before cancellation.
# Menu-Driven
1. Reservation
2. Cancellation
3. Exit
